# 🃏 War Card Game – Swift

A simple card-based **War Game** built using **Swift** and **UIKit**. This app simulates a two-player battle where each side draws a random card, and the higher card wins the round.

---

## 🛠️ Built With

- **Swift**
- **UIKit**
- **Xcode**
